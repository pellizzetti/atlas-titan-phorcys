## Getting Started

#### Requirements

Node:

- NodeJS >= 10.x
- NPM >= 6.x

Database:

- MySQL >= 5.6
- MariaDB >= 10.1
- PostgreSQL >= 10

#### Installation

```bash
yarn global add atlas-titan-phorcys

# OR npm install atlas-titan-phorcys -g
```

#### Create a new platform

```bash
atlas-titan-phorcys new my-platform
```

#### Start your platform

```bash
cd my-platform
atlas-titan-phorcys start
```
